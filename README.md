# tetris
本项目实现了一个基于 C 语言的俄罗斯方块游戏模拟器。游戏包括标准七种方块类型（I、T、O、J、L、S、Z），支持顺时针旋转、多方块下落、碰撞检测以及行满消除逻辑。程序通过结构体'position'管理当前与下一个方块的位置与旋转状态，使用二维数组'occupied'记录已固定方块，自动检测行满并计算分数（单行 100 分，双行 300 分，三行 500 分，四行 800 分）。游戏支持自动下落到最低位置的策略，结合旋转判定确保方块合法放置，同时监控死亡判定线以实现游戏结束判定。输入格式为方块序列字符，输出当前方块旋转角度、水平坐标及累计分数。通过该项目，用户可体验俄罗斯方块的核心玩法，同时提供自动下落和决策的策略。
## 功能
- 自动下落到最低位置  
- 碰撞检测，防止方块重叠或越界  
- 顺时针旋转功能  
- 检测行满并自动消除，同时计分  
- 死亡判定线，游戏结束判定  

## 编译与运行
确保系统已安装 GCC 编译器（Windows 或 Linux 可用）：

```bash
gcc 俄罗斯方块.c -o tetris.exe
./tetris.exe

## 输入示例
I T O J L S Z
- 每个字符代表一个方块形状，顺序依次生成  
- `X` 代表结束输入  

## 输出示例
0 4 0
90 3 100
180 5 400
E
- 输出为 当前方块旋转角度、水平坐标、累计分数  
- `E` 表示游戏结束
